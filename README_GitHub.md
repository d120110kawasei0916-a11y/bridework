# 管理ツール（GitHub Pages 最小構成）

この2ファイルを **新リポジトリ `kanri-tool` の直下** に置いて、GitHub Pages を有効化してください。  
スマホは **URLを押すだけで開けます（QR不要）**。

## 手順
1. リポジトリ作成： `kanri-tool`  
2. この `index.html` をアップロード → Commit  
3. リポの **Settings → Pages** を開く  
   - Source: **Deploy from a branch**  
   - Branch: **main** / **/(root)**  
   - Save  
4. 数十秒後に公開URLが有効になります：  
   **https://d120110kawasei0916-a11y.github.io/kanri-tool/**

## iPhoneでワンタップ起動（ホーム画面に追加）
1. Safariで **https://d120110kawasei0916-a11y.github.io/kanri-tool/** を開く  
2. 共有（□↑）→ **ホーム画面に追加**
